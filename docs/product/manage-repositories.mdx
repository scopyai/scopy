title: "Manage repositories"
description: "Enable, disable, search, and sync repositories in a workspace."
---

Scopy only runs on repositories that are enabled inside the workspace.

## During onboarding

After connecting GitHub, Scopy shows the repositories available from the GitHub App installation. You can:

- Search repositories by name.
- Select all available repositories.
- Enable or disable individual repositories.
- Continue to the dashboard after saving the selection.

If no repositories are shown, check the GitHub App installation settings and confirm that at least one repository is granted to the app.

## Repository dashboard

The repository page lists repositories for the selected workspace. Enabled repositories are shown first, followed by disabled repositories.

From this page you can:

- Search by repository name or full name.
- Enable or disable review for a repository.
- Sync repositories from GitHub.
- Open a repository to view pull requests and review results.

## Syncing repositories

Use repository sync after changing GitHub installation access or when repository metadata appears stale. Sync pulls the latest accessible repositories from GitHub into the workspace.

## Archived repositories

Archived repositories are shown with reduced emphasis and cannot be toggled like active repositories. If a repository should be reviewed again, unarchive it in GitHub first and then sync the workspace.
